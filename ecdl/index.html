ECDL

During High School, after completing the ECDL course and after passing all the exams, I achieved the European Computer Driving License. It is a worldwide-recognized computer literacy certification program that according to the definition given to the main website, it provides students with the IT skills and abilities needed to build on existing knowledge, motivate further learning, and improve their employability in a wide range of fields and industries.

The modules I took are:

  1. Online Collaboration
  2. IT Security
  3. Using Databases
  4. Presentation
  5. Online Essentials
  6. Spreadsheets
  7. Word Processing
  8. Computer Essentials
  9. IT Security
